# poke-api

Uma API RESTful de Pokémons

## Pré requisitos

- Git (https://git-scm.com)
- Node (https://nodejs.org)
- MongoDB (https://www.mongodb.com)

## Clonando o repositório

```
git clone https://github.com/gabriel-me/poke-api.git
```

## Instalando dependências

```
cd poke-api
```

```
npm install
```

## Iniciando MongoDB

```
mongod
```

## Iniciando aplicação

```
npm run start
```
